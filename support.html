常见问题

🙏 祷告内容会被保存吗？

是的，您的祷告会被安全地保存在您的账户中，方便您随时查看历史记录。您可以：

  • 在"历史"页面查看所有祷告记录
  • 随时删除不需要的祷告
  • 导出您的祷告记录
  • 通过设置页面删除所有数据
🤖 AI回应是如何生成的？

我们使用先进的AI技术分析您的祷告内容，然后从圣经中选择相关的经文和智慧指导：

  • 基于圣经公有领域版本（避免版权问题）
  • 提供适合的经文引用和解释
  • 包含80-120字的智慧指导
  • 在紧急情况下提供专业资源链接

注意：AI回应仅供参考，不能替代专业的牧师咨询或心理健康服务。

🔒 我的隐私安全吗？

我们非常重视您的隐私和数据安全：

  • 加密传输：所有数据通过HTTPS加密传输
  • 安全存储：数据存储在SOC 2认证的安全数据中心
  • 访问控制：只有您可以访问自己的祷告内容
  • 不会出售：我们永远不会出售您的个人信息
  • 可删除：您可以随时删除账户和所有数据

详细信息请查看我们的隐私政策。

📱 支持哪些设备？

上帝说话支持以下设备：

  • Android：Android 8.0 (API 26) 及以上版本
  • iOS：iOS 13.0 及以上版本
  • 设备要求：至少2GB RAM，100MB存储空间
  • 网络：需要稳定的网络连接
💳 如何管理我的订阅？

订阅管理通过您的应用商店账户进行：

  • Android：Google Play 商店 → 我的应用和游戏 → 订阅
  • iOS：设置 → Apple ID → 订阅
  • 取消：至少在当前周期结束前24小时取消
  • 退款：按照各应用商店的退款政策执行
🌍 支持多种语言吗？

目前支持的语言：

  • 中文（简体） - 主要语言
  • 英语 - 完整支持
  • 其他语言 - 计划中，敬请期待

AI回应会根据您输入的语言自动匹配相应的圣经版本。

故障排除

应用无法启动

  1. 确保设备满足最低系统要求
  2. 检查网络连接是否正常
  3. 重启应用或重启设备
  4. 更新到最新版本
  5. 如问题持续，请联系客服

无法获得祷告回应

  1. 检查网络连接
  2. 确认是否已达到月度使用限制
  3. 尝试重新输入祷告内容
  4. 避免输入过短或过长的内容
  5. 稍后重试，服务器可能暂时繁忙

登录问题

  1. 确认邮箱和密码正确
  2. 检查邮箱是否已验证
  3. 尝试"忘记密码"重置
  4. 清除应用缓存
  5. 联系客服协助恢复账户
